# Week 2: Search, Filter & Category Enhancement

## Objective
Improve content discoverability through advanced search and filtering.

## Summary
In Week 2, advanced search and filtering mechanisms were implemented to help users easily find monuments. Filters based on location, category, and historical era were added, improving usability and navigation within the app.

## Key Activities
- Implemented search functionality for monuments  
- Added category-based and location-based filters  
- Optimized UI for faster content discovery  

## Outcomes
- Improved monument search and filtering  
- Enhanced user navigation experience
