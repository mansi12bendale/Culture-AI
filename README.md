# CultureAI: Showcasing the Rich Cultural Heritage and Traditions of India

## Project Overview

**CultureAI** is a mobile app designed to showcase India's rich cultural heritage, featuring heritage sites, temples, forts, caves, and traditions. The app provides historical descriptions, multimedia content, location-based navigation, and user interaction features (such as reviews and ratings).

### Target Domains:
- Mobile Application Development (Android)
- Digital Heritage and Tourism
- Cultural Studies and Preservation

## Key Features:
- Explore heritage sites through images, videos, and descriptions.
- Google Maps integration for navigation.
- User reviews and ratings for engagement.
- Push notifications for heritage events and featured updates.
- Offline access to crucial information via SQLite.

## Phase 1: Project Conceptualization and Planning

This phase involved defining the application's goals, identifying key features, and outlining the technical approach.

### Application Overview:
- **Mobile app** as a digital guide to India's heritage sites, temples, forts, caves, and traditions.
- Historical descriptions and multimedia content.
- Location-based navigation using **Google Maps**.
- User interaction with **reviews** and **ratings**.
  
---

## Phase 2: Data Acquisition, Preprocessing, and Integration

This phase focused on collecting resources and preparing the database.

### Data Acquisition:
- Curated a list of Indian heritage sites, temples, forts, caves, and monuments.
- Gathered high-quality **images** and **videos** for each heritage site.
- Collected historical and cultural descriptions.

### Preprocessing:
- Organized multimedia files for efficient retrieval.
- Standardized historical data for consistent presentation.
- Structured site metadata for easy search and map display.

### Integration:
- Chose **Firebase** for cloud storage (for real-time updates).
- Used **SQLite** for offline data storage to ensure seamless user experience.

---

## Phase 3: Application Development, Testing, and Evaluation

Focused on building and testing the mobile application.

### Frontend Development:
- Developed using **Java** in **Android Studio**/VS Code.
- Designed an intuitive and interactive User Interface (UI).
- Integrated **Google Maps API** for location-based navigation.

### Backend Development:
- Set up **Firebase** for user authentication, storing reviews, ratings, and multimedia content.
- Implemented **SQLite** for offline data access.

### Multimedia Handling:
- Loaded **images** and **videos** from **Firebase Storage** and **YouTube API**.

### User Interaction Features:
- Built a review and rating system stored in Firebase.
- Developed a **push notification system** to alert users about events and updates.

### Testing:
- Conducted extensive testing across Android devices.
- Ensured smooth functionality of **maps**, **offline mode**, and multimedia display.
